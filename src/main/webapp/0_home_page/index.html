内蒙古伊泰煤制油有限公司200万吨/年煤炭间接液化示范项目是由内蒙古伊泰煤制油有限责任公司实施建设的大型煤炭间接液化示范项目。项目已完成可行性研究报告并通过石油化工规划院组织的专家审查。
